# 00_指南使用说明

## 指南目标

本指南用于帮助 RoboMaster 队伍完成经验传承、工作规范沉淀和下一届队员培养。它应当回答三个问题：

- 队伍如何协作？
- 具体事务如何执行？
- 前人踩过的坑如何避免？

## 适用对象

- 队长、副队长
- 机械、电控、算法、运营等各组组长
- 兵种负责人
- 正式队员和新队员
- 上一届答疑成员

## 内容层级

本指南分为三类内容：

| 类型 | 作用 | 示例 |
| --- | --- | --- |
| 制度 | 明确职责和规则 | 会议纪律、延期处理、组长职责 |
| 流程 | 指导具体事务 | 采购申请、视频拍摄、赛前检查 |
| 模板 | 直接复制使用 | 会议纪要、设计评审、交接文档 |

## 维护规则

1. 每个章节必须有负责人。
2. 重要流程变更需要队长或相关组长确认。
3. 章节内容应尽量写成可执行清单，避免只写感想。
4. 每次赛季结束后，应集中更新一次本指南。
5. 重要事故、失败经验、临场问题必须进入复盘库。
