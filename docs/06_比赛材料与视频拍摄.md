# 06_比赛材料与视频拍摄

## 核心原则

机器人负责人不是造完车就结束，而是需要负责到视频拍摄完成、材料提交完成、问题复盘完成。

## 规则更新处理

每次组委会更新规则或参赛要求后，应立即完成：

1. 队长和组长阅读规则更新。
2. 标记影响报名、制作规范、视频、文档、时间节点的内容。
3. 拆解材料撰写和视频拍摄任务。
4. 明确负责人、截止时间和验收标准。
5. 不要等到提交截止日期才开始准备。

## 视频拍摄流程

```text
确认规则要求 -> 制定拍摄脚本 -> 检查机器人状态 -> 准备场地和物资 -> 拍摄 -> 初审 -> 补拍 -> 归档 -> 提交
```

## 拍摄人员要求

- 兵种机械负责人应在场。
- 兵种电控负责人应在场。
- 涉及自瞄、导航时，算法负责人应在场。
- 操作手、摄像、物资负责人提前确认。
- 如果某个兵种已经具备拍摄条件，应优先拍摄，不要等待所有兵种一起完成。

## 拍摄前检查

- 电池电量和备用电池。
- 弹丸、工具、电脑、网线、充电器。
- 机器人机械结构是否可靠。
- 电控是否能稳定启动。
- 自瞄是否能正常识别。
- 红蓝方设置是否正确。
- 视频要求的动作是否逐项覆盖。

## 视频归档

每个视频应记录：

```text
视频名称：
对应规则条款：
负责兵种：
负责人：
拍摄日期：
最终文件链接：
是否已提交：
审核人：
```
