# 07_采购预算与报销

## 采购原则

采购不是简单买东西，而是为研发、调试和比赛交付提供资源。关键硬件应说明不购买的后果和替代方案。

## 采购申请方式

采购申请统一通过飞书审批功能完成，不单独维护 Markdown 采购申请模板。飞书审批表单应作为唯一正式申请入口，避免同一事项在多个地方重复填写。

飞书审批中建议保留这些字段：

- 物品名称、型号、数量、单价、总价、购买链接。
- 用途、使用模块、负责人、预计使用时间。
- 为什么必须购买。
- 不购买的后果。
- 替代方案。
- 是否可复用。
- 审批人和审批结论。

## 审批建议

| 类型 | 审批人 |
| --- | --- |
| 普通耗材 | 组长 |
| 关键硬件 | 组长 + 队长 |
| 高金额设备 | 队长 + 指导老师/财务负责人 |
| 紧急采购 | 队长确认后补材料 |

## 硬件选型论证

自瞄、导航、小电脑、相机、镜头等关键设备应进行选型比较。

比较维度：

- 性能是否满足需求
- 价格
- 供货周期
- 队伍已有经验
- 软件和驱动支持
- 可维护性
- 可复用性

## 发票与报销

待补充：根据学校或学院要求填写具体流程。

至少应明确：

- 发票抬头
- 税号
- 报销材料
- 提交人
- 提交截止时间
- 纸质材料保存位置

## 物资管理

- 贵重设备应登记负责人。
- 相机、小电脑、电池、遥控器、工具等应有借用记录。
- 损坏、进水、丢失必须记录。
- 赛季结束后进行一次物资盘点。
