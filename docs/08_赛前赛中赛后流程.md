# 08_赛前赛中赛后流程

## 赛前一个月

- 检查规则和提交材料要求。
- 明确所有机器人剩余问题。
- 确定视频拍摄计划。
- 检查关键物资是否采购完成。
- 进行完整形态测试。
- 安排操作手训练。

## 赛前一周

- 检查所有机器人机械、电控、算法状态。
- 检查电池、弹丸、工具、备用件。
- 确认比赛时间、交通和住宿。
- 完成赛场物资打包。
- 确认每个成员职责。

## 上场前检查

```text
红蓝方阵营是否正确：
自瞄是否攻击敌方：
是否误识别队友：
超电离线后状态机是否正常：
电池开关保护是否牢固：
电池电量是否充足：
弹丸是否准备好：
操作手是否到位：
备场人员是否到位：
```

## 赛中物资管理

- 电池和弹丸应由专人看管。
- 备场工具箱应固定负责人。
- 每场后记录机器人损伤和异常。
- 不要让关键设备散落在不同成员手中。

## 赛后复盘

复盘应覆盖：

- 战术与操作
- 机器人可靠性
- 自瞄、导航、通信表现
- 备场和物资管理
- 临场决策
- 下一届改进建议
