# 09_招新培训与考核

## 培训原则

培训以自学为主，队伍的重点应放在答疑、作业、项目实践和开拓眼界上。

## 招新流程

待补充：

- 宣讲
- 报名
- 初筛
- 分方向培训
- 阶段作业
- 项目考核
- 正式队员选拔

## 机械方向培养

待补充：推荐软件、基础知识、练习项目、设计评审要求。

## 电控方向培养

建议路径：

1. 学习 STM32 基础外设。
2. 学习 HAL 库开发。
3. 完成 CAN 通信。
4. 控制达妙/大疆电机。
5. 制作简单云台。
6. 制作底盘控制。
7. 学习 PID 调参和上位机波形观察。
8. 学习经典控制理论和现代控制理论基础。

## 算法方向培养

待补充：环境配置、视觉基础、模型训练、部署、相机与镜头配置、调试记录。

## 阶段考核

考核应尽量项目化，不只看笔试或口头汇报。

```text
考核任务：
考核目标：
提交材料：
演示要求：
评分标准：
截止时间：
负责人：
```
