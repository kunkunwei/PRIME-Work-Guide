# 10_传承与交接

## 传承目标

传承的目标不是让下一届重新问一遍，而是让他们可以在已有资料上继续推进。

## 队长交接

应交接：

- 队伍组织架构
- 指导老师和学院沟通方式
- 预算和经费情况
- 报名、材料、账号权限
- 当前主要风险
- 下一赛季建议

## 组长交接

应交接：

- 本组成员情况
- 当前项目状态
- 已完成资料
- 未完成任务
- 常见问题
- 推荐学习路线
- 下一届优先事项

## 兵种负责人交接

应交接：

```text
兵种名称：
当前状态：
机械问题：
电控问题：
算法问题：
调试方法：
常见故障：
相关资料：
下一届建议：
```

## 资料交接

- 飞书空间权限
- GitHub 仓库权限
- CAD、代码、模型、数据集
- 采购和报销资料
- 视频和比赛材料
- 账号、密码、设备借用记录

## 答疑机制

上一届成员应尽量把答疑结果整理回文档，避免同一个问题被重复问。
